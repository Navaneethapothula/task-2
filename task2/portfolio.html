Projects

Detecting Sybil Attacks using Proofs of Work and Location of VANETs

The project proposes a Sybil attack detection scheme using proofs of work and location. Each road side unit (RSU) issues a signed time-stamped tag as a proof for the vehicle's anonymous location. The scheme requires multiple RSUs to issue trajectories, requiring an infeasible number of RSUs to create fake trajectories. The vehicle must solve a computational puzzle before obtaining a proof of location. During reported events, the event manager uses a matching technique to identify Sybil trajectories. Extensive experiments and simulations show a high detection rate with low false negative and acceptable communication and computation overhead.
